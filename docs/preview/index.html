Agent Graph and Message Flow

Implementation status: Stub-based agents and deterministic tool dispatch (no live LLM API calls in the current demo).

Agent Roles

Planner
Creates deterministic, dependency-aware step plans from goal text.
Worker
Runs each planned step through named tools in the registry.
Reviewer
Captures review traces and approval outcomes for sensitive actions.
Orchestrator
Schedules ready steps, applies retries/fallbacks, and persists run state.
